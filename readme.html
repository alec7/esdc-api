Erigones SDDC API¶

Erigones SDDC API Python Library.

  • Supported Python versions: >= 2.6 and >= 3.2

Installation¶

pip install erigones-sddc-api

Dependencies:

  • requests

Usage¶

>>> from erigones_sddc_api import Client
>>> es = Client(api_url='https://my.erigones.com/api', api_key='<your-api-key>')
>>> es.get('/vm').content
